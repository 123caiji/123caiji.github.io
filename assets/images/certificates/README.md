## 使用说明

### 图片要求
- **格式**: 支持JPG、PNG格式
- **尺寸**: 建议800x600像素以上
- **大小**: 控制在1MB以内
- **命名**: 使用英文命名，便于管理

### 隐私处理
- 已对证书图片进行隐私保护处理
- 隐藏个人敏感信息（如身份证号）
- 仅展示证书名称、获奖等级、颁发时间等公开信息

### 更新维护
当获得新证书时：
1. 将证书图片放入此目录
2. 按命名规范重命名文件
3. 更新相关HTML页面
4. 更新此说明文档
